# 국방안테나공학 (Defense Antenna Engineering)

충남대학교 전파정보통신공학과 대학원 | 2026년 1학기

## 프로젝트 목록

### Project 1 - 안테나 원거리장 패턴 및 링크버짓
- 파일: `project1_antenna_pattern.m`, `project1_link_budget.m`
- 내용: 다이폴 안테나 방사 패턴 시뮬레이션, Friis 전송 방정식 기반 링크버짓 계산

### Project 2 - 원형편파 성능 및 근접장 측정
- 파일: `project2_far_field.m`, `project2_CP_performance.m`
- 내용: 근접장 스캔 데이터로부터 PWS 변환 → 원거리장 재구성, Back-Projection 알고리즘

### Project 3 - 빔포밍 배열안테나
- 파일: `project3_beamforming.m`
- 내용: 균일 선형 배열(ULA) 빔포밍, Chebyshev/Taylor 가중치 적용

### 기말 - Infinitesimal Dipole Modeling (IDM)
- 파일: `advanced_dipole_modeling.m`
- 내용: IDM 기반 근접장 측정 보정, Back-Projection 고도화
